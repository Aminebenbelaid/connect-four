# Connect four

$ bin/connect_four

## Setup

```shell
cd connect_four
gem install bundler
```

Install ruby-3.2.3 and ensure version with:

```shell
ruby -v # => 3.2.3
```

Install dependencies with:

```shell
bundle
```

## Test

```shell
rake test
```

## Development

A test driven approach is appreciated.

Tests should be written with `minitest/spec` syntax using minitest assertions.
